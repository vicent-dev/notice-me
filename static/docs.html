get /api/notifications
Get notifications (apiNotificationsGet)
Retrieves a paginated list of notifications.

Query parameters

pageSize (optional)
Query Parameter — Number of notifications per page.
page (optional)
Query Parameter — Page number.

Return type

Pagination

Example data

Content-Type: application/json
{
  "total" : 0,
  "pageSize" : 1,
  "page" : 6,
  "items" : [ {
    "clientId" : "clientId",
    "clientGroupId" : "clientGroupId",
    "id" : "id",
    "body" : "body",
    "notifiedAt" : "2000-01-23T04:56:07.000+00:00"
  }, {
    "clientId" : "clientId",
    "clientGroupId" : "clientGroupId",
    "id" : "id",
    "body" : "body",
    "notifiedAt" : "2000-01-23T04:56:07.000+00:00"
  } ]
}
